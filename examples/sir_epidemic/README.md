# SIR Epidemic Model

This example implements a simple spatial SIR epidemic model with ABMForge.

## Purpose

The model demonstrates how local contact between agents can produce epidemic diffusion over time.

## ABM concepts

- Agent state transitions
- Spatial contact
- Local contagion
- Random activation
- Model-level recording
- Reproducible scenario runs

## How to run

```bash
python examples/sir_epidemic/run.py
