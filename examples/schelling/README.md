# Schelling Segregation Model

This example implements a classic Schelling segregation model with ABMForge.

## Purpose

The model demonstrates how local preferences can produce emergent spatial segregation.

## ABM concepts

- Spatial agents
- Grid-based interaction
- Neighborhood effects
- Random activation
- Emergent macro-patterns
- Model-level recording

## How to run

python examples/schelling/run.py
