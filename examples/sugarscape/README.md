# Sugarscape

This example implements a simplified Sugarscape model using ABMForge.

## Purpose

Sugarscape is one of the most influential agent-based models.

Agents move across a landscape, collect resources, and accumulate wealth.

## Concepts

- Resource landscape
- Agent movement
- Resource collection
- Wealth accumulation
- Emergent inequality

## Run

    python3 examples/sugarscape/run.py

## Output

    outputs/sugarscape/

## Key APIs

- GridWorld
- RandomActivation
- Scenario
- Dataset
