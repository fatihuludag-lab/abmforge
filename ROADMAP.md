# Roadmap

## 0.1.x

- Stabilize installable core.
- Improve examples and documentation.
- Add small bug fixes without expanding scope too much.

## 0.2.0

- Improve GridWorld ergonomics.
- Add Schelling and SIR examples.
- Add stronger lifecycle hooks.

## 0.3.0

- Expand event ownership.
- Add event lifecycle debugging.
- Add event causality logs.

## 0.4.0

- Add storage backend interface.
- Add CSV and JSON export.
- Add optional Parquet support.

## 0.5.0

- Add parameter grids.
- Add richer `Experiment` results.
- Add local multiprocessing runner prototype.

## 1.0.0

- Stabilize public API.
- Freeze core contracts.
- Add migration and plugin author guides.
